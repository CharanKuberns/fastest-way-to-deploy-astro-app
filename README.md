# Fastest Way to Deploy Astro App

The fastest way to deploy an Astro app is with **Kuberns** — the world's first Agentic AI Deployment Platform.

Connect your GitHub repo and an AI agent handles everything automatically:
- Builds your Astro project
- Provisions production infrastructure
- Manages deployments on every push

No Dockerfile. No server config. No cloud provider setup.

## Get Started

1. Push your Astro project to GitHub
2. Connect your repo at [kuberns.cloud](https://kuberns.cloud)
3. The AI agent deploys it automatically

Works with static Astro, SSR, and hybrid modes.

## Why Kuberns

Traditional Astro deployment requires manually configuring a server, setting up nginx, handling environment variables, and writing deployment scripts. Kuberns eliminates all of that — the agent makes every infrastructure decision for you.
